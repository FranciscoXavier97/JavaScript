Curso de JavaScript 

Javascript es un lenguaje interpretado el cual puede ser ejecutado y entendido por navegadores web. Por ello es uno de los lenguajes mas utilizados en los navegadores. Se utiliza en el Frontend y Backend con tecnologias como node.js y frameworks como es Angular,React,Vue.js.

En este curso se empezo desde 0 asi que instalamos la herramienta conocida como visual studio code nos permitira trabajar mas facilmente con el lenguaje de JavaScript.

Tambien instalamos Node.js nos permitira ejecutar los programas en tiempo real  que hagamaos de javascript.

Para llevar este curso instalamos algunas extensiones, asi que dando click en el boton de extensiones empezamos a buscarlas

1.- Quokka.js nos permite agilizar el desarrollo de neustras aplicaciones de javascript sobre todo cuando estamos empezando con este lenguaje de programacion, asi que simplemente con salvar el codigo vamos a visualizar el resultado en consola. Con la version gratuita es mas que suficiente.

2.-Bracket Pair Colorizer y seleccionamos la version 2, esta xtension nos permite ver rapidamente en que momento estamos abriendo y cerrando ya sea un parentesis un corchete o una llave.

3.-ESlint nos permite agregar las mejeores practicas a nuestro codigo y tambien agregar formato al codigo que estemos escribiendo.

4.-Prettier - Code formatter 

5.-Debugger for Chrome nos permite trabajar directamente con visual studio code sin necesidad de salir de nuestra herramienta y ya no tenemos que ir a chrome para hacer debug.

6.-Visual Studio IntelliCode nos permite completar el codigo de manera inteligente cuando estamos escribiendo de manera automatica.

7.-Path Intellisense nos permite seleccionar mas facilmente los archivos, aunque esten distintas carpetas vamos a poder seleccionar mas facilmente los recursos que compartamos en nuestras paginas HTML.
